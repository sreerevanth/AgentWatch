# AgentWatch

AgentWatch is a reliability, safety, and observability layer for AI agents. It
watches agent execution, scores the risk of the actions agents take, and gives
you a CLI to inspect sessions, review safety, and report on cost.

This documentation focuses on the **command-line interface**.

- **[Getting Started](cli/getting-started.md)** — install AgentWatch and run your first commands.
- **[CLI Commands](cli/commands.md)** — a reference for every command group.
- **[CLI API Reference](cli/api.md)** — auto-generated API docs for the CLI helper modules.
